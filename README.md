# Archivos de Configuración
Archivos de Configuración de distintas herramientas usadas: Vim, Git, etc.

Los archivos se pueden distribuir opcionalmente de la siguiente manera:

~/.bashrc

~/.bash_profile

~/.gitconfig

~/git-completion.sh

~/git-prompt.sh

~/.vim/vimrc

~/.vim/plugin/acp.vim

~/.vim/autoload/acp.vim
